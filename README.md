# 🌌 CMB Temperature Analysis & Visualization Lab

Cosmic Microwave Background (CMB) Temperature Analysis &amp; Visualization Lab A Python-based physics simulation project that analyzes and visualizes the temperature fluctuations of the Cosmic Microwave Background — the afterglow of the Big Bang — using scientific computation and data visualization techniques.

---

## 🔭 About the Project

The **Cosmic Microwave Background (CMB)** is one of the strongest pieces of evidence for the Big Bang theory.  
This project simulates and visualizes tiny temperature variations in the CMB that reveal the early structure of the universe.

---

## ✨ Features

- 🌡️ CMB temperature fluctuation simulation
- 📊 Heatmap visualization of sky data
- 🧠 Physics-based modeling of early universe radiation
- 📈 Power spectrum / statistical analysis (optional extension)
- 🎨 Beautiful matplotlib-based scientific plots

---

## 🧪 Physics Concept

This project is based on the:
- Big Bang cosmology
- Photon decoupling era
- Blackbody radiation
- Temperature anisotropies in the CMB
